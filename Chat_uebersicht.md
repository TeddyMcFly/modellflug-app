# Chat-Uebersicht

Diese Datei hilft dabei, die verschiedenen Codex-Chats zum Projekt `modellflug_app` auseinanderzuhalten.

## Nutzung

- Pro Chat eine Zeile eintragen.
- Die Spalte "Anfragen" meint: Wie viele Nachrichten/Fragen du in diesem Chat ungefaehr gestellt hast.
- Wenn ein Chat sehr gross wird, lieber einen neuen Chat anfangen und hier notieren.
- Wichtige Absprachen kurz in die Spalte "Notiz" schreiben, damit andere Chats sie nachvollziehen koennen.

## Offene Hinweise

- Vor neuen Aenderungen zuerst `AGENTS.md`, Projektstruktur und Git-Status pruefen.
- Nicht mehrere Chats gleichzeitig an denselben Dateien arbeiten lassen.
- Wenn ein Chat eine wichtige Entscheidung trifft, hier kurz eintragen.
