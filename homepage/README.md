# Modellflug-Heaven Homepage

Dieser Ordner ist die getrennte Info-Homepage.

Die Flutter-App bleibt im Projekt weiter unveraendert unter `lib/`, `web/` und
`build/web`. Diese Homepage ist eine einfache statische Webseite und kann spaeter
als eigene Firebase-Hosting-Seite veroeffentlicht werden.

Lokale Vorschau bequem starten:

```powershell
.\Start.cmd
```

Das Skript startet den kleinen Server und zeigt diese Adresse an:

```text
http://localhost:52800
```

Falls die Homepage schon laeuft, wird nichts doppelt gestartet. Dann wird nur die
Adresse erneut angezeigt.

Geplanter Programm-Link:

```text
https://flugbuch.modellflug-heaven.de
```
